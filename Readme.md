# 📊 Customer & Sales Insights Dashboard

## 🔍 Project Overview

This project analyzes customer behavior and sales trends using Excel. It includes data cleaning, transformation, and interactive dashboard creation.

---

## 📁 Dataset

* E-commerce retail dataset
* Cleaned by removing null values, cancellations, and irrelevant entries

---

## 📊 Dashboard Features

* Top 10 Customers by Sales
* Sales by Country
* Monthly Sales Trend
* Top 10 Products

---

## 🎯 Key Insights

* Top customers contribute a major share of revenue
* United Kingdom dominates overall sales
* Sales peak observed in October–November
* Few products drive the majority of revenue

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers
